## 移动端 ui 样式库

### 通过 npm 进行安装

```
npm i app-beauty 
```
### 使用方式

#### 1.直接引用 css 文件
本插件依赖 font-awesome@4.7.0，安装本插件的同时也会安装 font-awesome。在使用本 UI 样式库的时候可以直接引用对应的 CSS 文件：

```
import 'font-awesome/css/font-awesome.min.css';
import 'app-beauty/css/ui.min.css';
```
#### 2.支持css模块化
在支持 CSS 模块的项目里也可以像下面这样直接引用插件：

```
import 'app-beauty';
```
